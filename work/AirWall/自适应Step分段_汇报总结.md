# 空气墙碰撞分段：弦高自适应步长方案

> 汇报用总结 · AirWall 项目 · 2026-06-04

## 一、要解决的问题

空气墙的视觉是一条平滑的 Spline 曲线，但碰撞必须用一串 **Box** 拼出来（引擎没有"曲线碰撞体"）。早期用**固定步长**分段：

```
段数 = Ceil(SplineLength / SegLen)，每段长度都是 SegLen
```

这种做法有两个硬伤：

| 问题 | 后果 |
|---|---|
| **直段浪费** | 一条笔直的墙也被切成几十个 Box，碰撞体数量爆炸、性能浪费 |
| **弯段穿漏** | 急转弯处 Box 是直的、墙是弯的，两者之间留缝，玩家能从缝里穿过去 |
| **末端外推 bug** | 最后一段 `D1=(i+1)*SegLen` 会超过曲线总长，取点函数沿切线"戳出去"一截 |

一句话：固定步长**该密的地方不够密，该疏的地方太密**。

## 二、核心思路：用"弦高"判断该切多细

借用几何里的 **弦高（sagitta）** 概念——一段曲线和它两端连成的直线（弦）之间的**最大缝隙**。

- **缝隙小** → 这段曲线接近直线 → 一个大 Box 就够，步子迈大点
- **缝隙大** → 这段弯得厉害 → 步子收小，多切几个 Box 贴合

这个缝隙的物理意义，正好就是"**碰撞盒和真实墙面之间最大漏多少**"，所以它直接当成**唯一的画质旋钮**：

```
sagitta = 距离( 曲线中点, 弦的中点 )
        = VSize( SplineMid − ChordMid )
```

其中 `SplineMid` 是曲线在该段中点的真实位置，`ChordMid` 是两端点连线的中点。两者差得越远，说明这段越弯。

## 三、算法结构：外层前进 + 内层细化

整个分段是**两层循环**：

### 外层 While —— 从头到尾走一遍曲线
- 条件：`(D < SplineLength) AND (guard < 4096)`
- 每段结束后 **`D = D1`** 前进到下一段起点
- `guard` 是防死循环的保命栓，最多跑 4096 段强制收手

### 内层 While —— 决定当前这段切多长
1. 先用**最大步长** `MaxStep` 试探，算出这段的 `sagitta`
2. 条件：`(sag > Tolerance) AND (step > MinStep)`——还太弯、且还能再砍，就继续
3. 每圈把 `step` **砍半**（二分），重算 `sagitta`
4. 直到 `sag` 达标 或 `step` 砍到 `MinStep` 触底 → 退出，接受当前段长

退出后这段的起点/终点已确定，进入 **EMIT** 环节生成 Box。

### EMIT —— 生成碰撞盒
```
Box 位置 = 段中点
Box 朝向 = FindLookAtRotation(起点 → 终点)
Box 尺寸 = SetBoxExtent(段长/2, 墙厚/2, 墙高/2)   ← Scale 必须 = 1
```

## 四、效果

| | 固定步长 | 弦高自适应 |
|---|---|---|
| 直段 | 切很多段 | **一步迈到底**，Box 数量大减 |
| 弯段 | 可能穿漏 | **自动加密**贴合曲线 |
| 末端 | 外推戳出去 | `D1=FMin(D+step, Len)` 天然夹断 |
| 调参 | 要同时调段长/数量 | **只调一个 Tolerance**（允许漏多少 cm） |

## 五、关键参数

| 参数 | 经验值 | 含义 |
|---|---|---|
| `MaxStep` | 400~800 cm | 直段单步最大长度，越大直段越省 |
| `MinStep` | 20~50 cm | 弯段最小步长，防止无限细分 |
| `Tolerance` | 5~15 cm | **唯一画质旋钮**：允许墙面最大漏缝，调它最直观 |

## 六、踩过的坑（汇报可选讲）

1. **死循环锁引擎**：外层 While 漏写 `D = D1` 自增 → 条件恒真 → Construction Script 一编译就把引擎 CPU 占满。UE 有 100 万次 Runaway Loop 保护伞会自动掐断，但循环体里有 AddComponent 时内存会暴涨，不能干等，得杀进程走资产隔离重建。**教训：自增和保命栓（guard、MinStep 触底）必须先连好再接执行线。**
2. **32 倍 Scale bug**：尺寸只能走 `SetBoxExtent`，`MakeTransform` 的 Scale 必须保持 1，否则 Box 被二次放大。
3. **搭建顺序**：先摆节点、连数据线、连好 `D=D1`，**最后**才把执行线接进 While 输入，避免搭建过程中预览实例 rerun 触发卡死。
