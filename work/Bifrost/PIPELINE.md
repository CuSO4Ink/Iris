## §2 等离子恒星（A1）技术方案

**分三层堆，从便宜到贵：**

1. **本体表面（主力，纯材质，最省）**：大球 mesh + 自定义材质。翻涌熔融感 = 多层 3D 噪声（domain warp 扭曲）+ 时间演化；emissive 拉出冷白过曝核心 + icy cyan 边缘梯度。
2. **isosurface 细节（可选加强）**：复用 NS_Slime marching cubes，仅在边缘轮廓 / 日珥弧处叠真 dynamic mesh，做"融合翻涌"立体感（呼应 `D3-KERNEL.md` A 方案"轮廓局部化"，不整球上，成本可控）。
3. **作为光源（因果链物理落点）**：恒星驱动方向光 / 大范围光 + 大气散射，真实照亮整个海岸。

**关键控制**：守 D2 冷神性红线——核心可过曝但保住 pale azure → icy cyan 色彩梯度，勿糊成纯白（LOG 2026-07-02 已记"控高光曝光"教训）。

## §3 天象（A2 极光 + A3 体积云）技术方案

**统一走一套 raymarching 体积框架**（TECH-SCOUT "管线 A—天空是一个体积场"），一套框架喂不同介质，避免重复造轮子：

| 天象 | 做法 | 成本 | 顺序 |
|---|---|---|---|
| A3 体积云 / 贝母云 | raymarch + 3D 噪声密度场，恒星光穿透散射 | 中 | **先做（最稳的底）** |
| A2 极光 | 同框架，改沿"磁力线"分布的发射介质 + 冷绿/冷紫，作恒星磁场可视化衍生 | 中高 | 后做 |
| god-rays / 大气散射 | 恒星光 + 体积雾自然产出 | 低 | 恒星衍生 |

> **因果链收束**：恒星光 → 云散射；恒星磁场 → 极光；恒星磁场 → 海面尖刺（§4）。天象与海面全是恒星的衍生，叙事高度自洽。

## §4 铁磁流体融合方案（点缀 · 融视觉不融物理）

**核心认知**：铁磁流体的震撼全在"尖刺阵列"视觉签名上，不在磁-流耦合物理。故**只偷视觉、不搬物理**，挂在 FluidFlux 基底上。

| 融合层级 | 做法 | 成本 | 采用 |
|---|---|---|---|
| ① 表面材质层 | FluidFlux 主波形 + 材质叠"磁尖刺" displacement/法线，恒星磁场"影响区"局部拉起规则尖刺 | 低 | ✅ 主力 |
| ② 局部 Niagara 点缀 | 恒星正下方"磁极点"放一小片真尖刺（Niagara + 局部高度场）当奇观焦点 | 中 | ✅ 焦点 |
| ③ 全海面真磁-流耦合 | 整片海铁磁流体求解 | 极高 | ❌ 已排除（与 FluidFlux 冲突 + 已复刻 FluidFlux） |

**采用 ①+②**：FluidFlux 主海面 → 材质层大范围"被磁场轻微扰动"尖刺质感（弱）→ 恒星正下方局部强尖刺奇观（Niagara）。
**收益**：保住"恒星磁场牵引海面长刺"因果链叙事，绕开 R&D ★★★ 磁-流耦合；成本从"10-15 天赌一把"降到"3-5 天稳拿视觉签名"。

## §5 场景布置 + MCP 量产管线（L1）

**围主机位⑤优化，不做全关卡。**

1. **构图 blockout 先行（最重要）**：灰模大块摆悬崖/遗迹/人物剪影/恒星的位置与比例，确认"渺小人 vs 压迫性巨构"构图成立——**先灰模定构图再做资产，避免返工。**
2. **kitbash 巨构**：岩石母材质 + 模块化遗迹件拼装，L1 走 MCP。
3. **景深分层**：前景（人物剪影+近岩）/ 中景（巨构遗迹）/ 远景（恒星+天象），资产精度递减，远景可低模。

### MCP 量产链路（P1 核心，也是 P3 要展示的工作流本身）
复用 UEAgent 的 `execute_python_script` / `execute_unreal_command`：
- **母材质生成**：定义参数面板规格（冷神性色域）→ 脚本批量建材质 + MI 变体 → 人 LookDev 验收
- **资产摆放**：脚本批量 spawn/transform 远景 mesh、植被散布 → 人调构图
- **规范铁律**（UE MCP 操作规范）：每批生成必设 **batch tag + semantic tags + Outliner folder + cleanup 方案**，保证可回滚可清理
