## 整体叙事主线

贯穿三段的一句话定位(建议作为作品集首页 slogan):

> "我能做出效果(Part 1)、能解释为什么好看(Part 2)、还能用 AI 让团队做得更快更新(Part 3)。"

每个项目统一用四要素呈现:**成品封面 → breakdown 拆解 → 性能数据 → 一句话技术总结**。
没有 breakdown 的项目不收录。

## Part 1 — 引擎资产效果功底

**要证明什么**:对传统引擎美术效果工作链路的完整、深入理解;不是调参,是懂底层 + 懂管线 + 懂预算。

> ⚠️ DyeSplashBaker / AirWall 是工作产出,**不直接收录**。下表"技术思路来源"列说明可借鉴的能力,但作品集项目须独立重做(换题材、自有版权)。

**收录项目(个人独立实现):**

| 个人项目(建议) | 定位 | 技术思路来源(工作经验背书) | 关键看点 |
|---|---|---|---|
| **程序化流体/腐蚀贴花 demo** | 个人题材的程序化贴花系统(如:岩浆蔓延 / 锈蚀扩散 / 魔法符文流淌) | DyeSplashBaker 的"Niagara 模拟→到达时间场→runtime 扫描"管线思路 | 完整管线 + 物理叙事 + RT 精度取舍 |
| **个人题材边界/能量场特效** | Spline 驱动、可交互的能量护盾/结界 | AirWall 的"编辑/碰撞/视觉/交互四层解耦"系统设计思路 | 系统设计 + 连续 UV + 接触反馈 + 预算意识 |

**核心价值主张:完整管线 + 性能意识 + 系统设计**,而非孤立炫酷效果。

**前沿技术衔接点(可选加分):**
- UE Substrate 分层材质(车漆/薄膜虹彩/SSS)
- Niagara Fluids / Data Channels
- 离线模拟 → runtime 代理(VAT / flipbook bake)

## Part 2 — 美术与审美能力

**要证明什么**:TA 的审美不是玄学,是可归纳、可量化的工程化能力;能把"更通透"翻译成"SSS + bloom 阈值"。

**收录素材(基于 AestheticGym 项目):**

- **审美四支柱产出**:光影 / 色彩 / 构图 / 材质,每支柱给 拆解笔记 + 复刻 demo
- **复刻对照**:挑 2~3 个经典效果做 1:1 复刻,旁边附"我观察到的视觉细节 + 用了什么技术还原"
- **物理反推审美**:用 BRDF / 色彩空间 / 渲染方程 反向当审美工具,展示"懂物理 → 调得准"
- **before/after 对比**:每个调整都有对照,讲清"改了什么、为什么变好看"

**这一段的核心价值主张:审美判断 + 视觉语言 + 沟通翻译三项可训练能力。**

## Part 3 — AI 融入 TA 工作流

**要证明什么**:前瞻性与稀缺性。2026 极少有 TA 作品集包含 AI 驱动管线,这是差异化天花板。

> ⚠️ UEAgent 是工作项目,**不直接收录**。其"自然语言操控 UE"的思路可作为能力背书,作品集须用个人独立搭建的版本演示(自有工程、自有题材)。

**收录项目(个人独立实现):**

| 个人项目(建议) | 定位 | 技术思路来源(工作经验背书) | 关键看点 |
|---|---|---|---|
| **个人 AI×DCC 工具 demo** | 自建 MCP/脚本,用自然语言驱动 UE/Blender 生成或修改资产 | UEAgent 的 MCP + Python 链路经验 | "说一句话 → 生成/改资产"端到端演示视频 |

**可扩展的 AI×TA 方向(按值得做排序):**

1. **AI 驱动 DCC 操作** — 个人自建,做成端到端演示(自然语言 → 蓝图/材质图/资产生成)
2. **AI 驱动效果生成** — 文本/图 → 材质、文本 → HDRI、生成式贴图嵌入工作流
3. **AI 驱动优化** — AI 辅助 LOD / 资产校验 / 性能瓶颈分析自动化
4. **神经渲染资产化** — 3DGS 资产化、Neural Texture Compression(高阶可选)

**核心价值主张:工具思维 + 前瞻视野,证明你是"定义未来工作流"的 TA。**
