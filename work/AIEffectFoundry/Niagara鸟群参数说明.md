# 鸟群参数说明

| 参数 | 作用 |
|---|---|
| **Show Leader** | 是否显示鸟群引导点。用于调试路径和跟随效果，正式使用时可关闭。 |
| **Crowd Scale** | 控制鸟的整体大小。数值越大，鸟越大；鸟变大后通常也需要更大的间隔。 |
| **Show Bound** | 是否显示鸟群包围盒。用于检查鸟群活动范围和是否被正确包住，正式使用时可关闭。 |
| **Crowds Num** | 控制鸟的数量。数值越大，鸟群越密，性能消耗也越高。 |
| **Spawn Radius** | 控制鸟出生时的散布范围。数值越大，出生时分布越散。 |
| **Crowds Speed** | 控制鸟群沿路径飞行的整体速度。数值越大，鸟群飞得越快。 |
| **Separation Weight** | 控制鸟之间互相避让的强度。数值越大，鸟越不容易贴在一起；过大可能显得很散或抖动。 |

## 调参建议

- 想让鸟群更大或更小，调 **Crowd Scale**。
- 想改变鸟的数量，调 **Crowds Num**。
- 想让鸟群飞快或飞慢，调 **Crowds Speed**。
- 想让鸟之间不要太黏，优先调 **Separation Weight**；如果有间隔参数，则优先调间隔参数。
- **Show Leader** 和 **Show Bound** 主要是调试显示，最终镜头里一般关闭。
