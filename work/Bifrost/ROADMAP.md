# Bifrost · ROADMAP（唯一落地指南）

> **本文件是项目唯一的执行主图**：读它就知道下一步做什么。其他文档只在被本文件引用时才读。
> **前提锁定**：D1=等离子恒星驱动天气 · D2=冷神性 · 主机位⑤=场景内最优截图点（非设计驱动因子）· 内核=A 方案（轮廓局部化）· UE5.8 · FluidFlux 已复刻。
> **场景定位（07-07 重构）**：第三人称跑图 diorama，200m 单向海滩走廊，30-60s 走完，扛得住玩家游览。

## §1 场景结构（4 段海滩走廊）

200m 单向海滩分 4 段，玩家从远端出生 → 沿沙滩前进 → 中景 → 主锚点区 → 回望端。两侧悬崖 + 海面作视觉边界，玩家不可达。

| 段 | 位置 | 视觉重心 | LookDev 精力占比 |
|---|---|---|---|
| **S1 远端** | 0-50m 出生点 | 远景巨构 + 恒星建立镜头 · 引导玩家往前走 | 15% |
| **S2 中景** | 50-120m 过渡带 | 遗迹 kitbash 群 · 海面 FluidFlux 主景 | 15% |
| **S3 主锚点区** | 120-160m | 主机位⑤最优截图点 · 铁磁流体尖刺 · 极光焦点 | **60%（精修）** |
| **S4 回望端** | 160-200m 尾段 | 玩家走过后回望远景 · 收束叙事 | 10% |

> **主图机位⑤的新定位**：在 S3 段某处，玩家走过时自然形成⑤号构图（可放个视觉提示"这里适合截图"）。主图不再驱动整个场景设计，而是场景做好后的"最优机位自然产物"。

**风险缓冲机制**：
- 周五半天缓冲：每周固定周五下午不排硬任务，用于回滚 / 找补 / 走位测试。
- W3 末保底成片是硬线，即使 W4-W5 全砍也已成片（无点缀 + 全场景基础 LookDev）。
- 铁磁流体降级为材质+局部 Niagara（详见 `PIPELINE.md` §4），3-5 天可控。
- **P4 LookDev/调色不可 MCP 提效，纯人力**，W3/W4 各留 2 天专属窗口。
- **玩家角色资产用户自处理**，本 ROADMAP 不排任务；T-P0.2 用 UE 默认 ThirdPersonCharacter 跑通即可。

## §5 关键任务验收颗粒度

### T-P0.0 路径设计（W1，0.5 天）
- 顶视图手绘 200m 海滩路径线 + 4 段边界 + 4 个视觉锚点位置 + 每个锚点机位朝向 + 玩家在此停留几秒
- 产物：一张顶视图 + 4 段 thumbnail
- 归档：`breakdown/P0-path-design/`

### T-P0.1 4 段海滩 blockout（W1，3.5 天）
下游所有 L1 量产依赖它，验收必须量化：
- **总长度** 200m 单向 · **宽度** 20-40m（沙滩宽度）· **两侧边界** 悬崖 mesh 或海面 Fluid（玩家不可越界）
- **每段 blockout 达成**：S1/S2/S3/S4 各段的低精代摆资产 blockout（用基础 cube/cylinder 代摆）
- **玩家路径可通行**：从 S1 出生点走到 S4 尾段无卡顿、无掉海、无穿墙
- **S3 主机位⑤构图**：站在 S3 中心时视野内构图达成原 T-P0.1 5 项量化验收：
  - 主机位 FOV 定死（24-35mm 等效）
  - 恒星屏占比（30-45%）
  - 人物剪影屏占比（3-8%）
  - 景深三档比例（前/中/远景分层）
  - 背景光比目标（恒星 vs 中景遗迹 EV 差）
- 灰模截图归档到 `breakdown/P0-blockout/`

**验收：以上全部达成，T-P0.1 才算完成。**

### T-P0.2 第三人称接入（W1，0.5 天）
- 拉 UE 默认 ThirdPersonCharacter 到场景，调 walk speed、camera height
- 禁用 jump（海滩不需要）
- **验收**：可从 S1 走到 S4，全程不掉海不穿墙，帧率不塌
- 玩家角色 mesh 后续用户自替换，本任务用默认小白人即可

### T-L2.3a S3 主锚点区精修 LookDev（W3 起启动，W4 定稿，60% 精力）
- **色域守则**：pale azure / icy cyan / 冷白高光；核心可过曝但保住色彩梯度不糊成纯白
- **调色链路可复现**：所有关键节点参数落文档，出成 preset
- **验收标准**：主机位⑤下渲染的成片满足 D2 定案提示词的视觉气质（对比 `CONCEPT.md` §3）
- 调色前后对比截图归档到 `breakdown/P4-lookdev-S3/`

### T-L2.3b 其余段基础 LookDev（W4，40% 精力）
- S1/S2/S4 各段快扫一遍，保证跑过时不塌方
- 不追求极致，颜色与 S3 一致即可
- **验收**：走位测试中每段截图 3 张，无一张观感塌方
