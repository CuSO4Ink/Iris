# DyeSplashBaker — FloodStep v2 方案（已归档）

> **状态**：已废弃，由 SWE Level 2 烘焙方案取代。
> **归档日期**：2026-04-20
> **归档原因**：FloodStep 本质是"各向异性测地距离场"，缺流体要素（时间演化、质量守恒、非线性反馈），扩散表现死板重复。
> **后续方案**：见 `work/DyeSplashBaker/SWE-TechSpec.md`

## 方案概述

三阶段 Scratch Pad 管线：
1. **Stage 1 — InitGrid**：在 Emitter Spawn 时设置种子区域值为 1.0，其余 0.0
2. **Stage 2 — FloodStep v2**：每帧用 `max(邻居 - step)` 反向传播，带 8 邻域噪声调制和方向性受力
3. **Stage 3 — NormalizeGradient**：把 `[MinGradientValue, 1.0]` 重映射到 `[0, 1]` 并做 Gamma 校正

## Stage 1：InitGrid

```hlsl
// ====== InitGrid — 粘贴到 Simulation Stage 0 的 Scratch Pad Custom HLSL ======
// 
// 模块输入参数（在 Scratch Pad 的 Input Map 中添加）：
//   Grid2DCollection  : Grid 2D Collection (Data Interface)
//   SeedCenter        : Vector 2D         (User Parameter)
//   SeedRadius        : Float             (User Parameter)
//
// Simulation Stage 设置:
//   Iteration Source = Grid2DCollection
//   Execute Behavior = On Emitter Spawn（只执行一次）

// 归一化 UV
int NumCellsX,NumCellsY;
G2D.GetNumCells(NumCellsX, NumCellsY);
float2 CellUV;
CellUV.x = (float(IndexX) + 0.5) / float(NumCellsX);
CellUV.y = (float(IndexY) + 0.5) / float(NumCellsY);

// 到种子中心的距离
float Dist = length(CellUV - SeedCenter);

// 种子区域 = 0（最先到达），其余 = 1（未到达）
GradientValue = (Dist <= SeedRadius) ? 1.0 : 0.0;
```

## Stage 2：FloodStep v2

```hlsl
// ============================================================
// FloodStep v2 (反转版) — 灰度从 1→0 递减
// 种子 = 1.0, 未到达 = 0.0, 传播用 max(邻居 - step)
// ============================================================
//
// 外部输入引脚：
//   Grid2D Collection:  G2D
//   Texture2D:          NoiseSampler
//   int:                IndexX, IndexY
//   float:              BaseStep
//   float:              NoiseScale
//   float:              NoiseIntensity
//   float:              NoiseOffset
//   float2:             ForceDirection
//   float:              ForceStrength
//   float:              ForceDecay
//   float2:             SeedCenter
//   float:              ArcScale          ← 对角系数，推荐 1.41421356
//
// 输出引脚：
//   float:              MaxCandidate
// ============================================================


float CurrentValue;
G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX, IndexY, CurrentValue);

// 种子点 = 1.0，不动
if (CurrentValue >= 1.0)
{
    MaxCandidate = 1.0;
    return;
}

int NumCellsX, NumCellsY;
G2D.GetNumCells(NumCellsX, NumCellsY);

float2 CellUV = (float2(IndexX, IndexY) + 0.5) / float2(NumCellsX, NumCellsY);

// ============================================================
// 读取 8 邻居（未到达的默认值 = 0.0）
// ============================================================
float N = 0.0, S = 0.0, W = 0.0, E = 0.0;
float NW = 0.0, NE = 0.0, SW = 0.0, SE = 0.0;

if (IndexY - 1 >= 0)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX, IndexY - 1, N);
if (IndexY + 1 < NumCellsY)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX, IndexY + 1, S);
if (IndexX - 1 >= 0)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX - 1, IndexY, W);
if (IndexX + 1 < NumCellsX)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX + 1, IndexY, E);
if (IndexX - 1 >= 0 && IndexY - 1 >= 0)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX - 1, IndexY - 1, NW);
if (IndexX + 1 < NumCellsX && IndexY - 1 >= 0)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX + 1, IndexY - 1, NE);
if (IndexX - 1 >= 0 && IndexY + 1 < NumCellsY)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX - 1, IndexY + 1, SW);
if (IndexX + 1 < NumCellsX && IndexY + 1 < NumCellsY)
    G2D.GetPreviousFloatValue<Attribute="GradientValue">(IndexX + 1, IndexY + 1, SE);

// ============================================================
// 方向噪声采样
// ============================================================
float2 NoiseUV = CellUV * NoiseScale;

float4 Smp_N, Smp_S, Smp_W, Smp_E;
float4 Smp_NW, Smp_NE, Smp_SW, Smp_SE;

NoiseSampler.SampleTexture2D(NoiseUV + float2( 0.0,          -NoiseOffset), 0.0, Smp_N);
NoiseSampler.SampleTexture2D(NoiseUV + float2( 0.0,           NoiseOffset), 0.0, Smp_S);
NoiseSampler.SampleTexture2D(NoiseUV + float2(-NoiseOffset,   0.0),         0.0, Smp_W);
NoiseSampler.SampleTexture2D(NoiseUV + float2( NoiseOffset,   0.0),         0.0, Smp_E);
NoiseSampler.SampleTexture2D(NoiseUV + float2(-NoiseOffset,  -NoiseOffset), 0.0, Smp_NW);
NoiseSampler.SampleTexture2D(NoiseUV + float2( NoiseOffset,  -NoiseOffset), 0.0, Smp_NE);
NoiseSampler.SampleTexture2D(NoiseUV + float2(-NoiseOffset,   NoiseOffset), 0.0, Smp_SW);
NoiseSampler.SampleTexture2D(NoiseUV + float2( NoiseOffset,   NoiseOffset), 0.0, Smp_SE);

float Raw_N  = Smp_N.r;  float Raw_S  = Smp_S.r;
float Raw_W  = Smp_W.r;  float Raw_E  = Smp_E.r;
float Raw_NW = Smp_NW.r; float Raw_NE = Smp_NE.r;
float Raw_SW = Smp_SW.r; float Raw_SE = Smp_SE.r;

float Avg = (Raw_N + Raw_S + Raw_W + Raw_E + Raw_NW + Raw_NE + Raw_SW + Raw_SE) * 0.125;
Avg = max(Avg, 0.001);

float NMod_N  = max(0.1, 1.0 + (Raw_N  / Avg - 1.0) * NoiseIntensity);
float NMod_S  = max(0.1, 1.0 + (Raw_S  / Avg - 1.0) * NoiseIntensity);
float NMod_W  = max(0.1, 1.0 + (Raw_W  / Avg - 1.0) * NoiseIntensity);
float NMod_E  = max(0.1, 1.0 + (Raw_E  / Avg - 1.0) * NoiseIntensity);
float NMod_NW = max(0.1, 1.0 + (Raw_NW / Avg - 1.0) * NoiseIntensity);
float NMod_NE = max(0.1, 1.0 + (Raw_NE / Avg - 1.0) * NoiseIntensity);
float NMod_SW = max(0.1, 1.0 + (Raw_SW / Avg - 1.0) * NoiseIntensity);
float NMod_SE = max(0.1, 1.0 + (Raw_SE / Avg - 1.0) * NoiseIntensity);

// ============================================================
// 受力方向调制
// ============================================================
float2 Dir_N  = float2( 0.0, -1.0);
float2 Dir_S  = float2( 0.0,  1.0);
float2 Dir_W  = float2(-1.0,  0.0);
float2 Dir_E  = float2( 1.0,  0.0);
float2 Dir_NW = normalize(float2(-1.0, -1.0));
float2 Dir_NE = normalize(float2( 1.0, -1.0));
float2 Dir_SW = normalize(float2(-1.0,  1.0));
float2 Dir_SE = normalize(float2( 1.0,  1.0));

float ForceLen = length(ForceDirection);
float2 ForceDir = (ForceLen > 0.001) ? (ForceDirection / ForceLen) : float2(0.0, 0.0);

float DistFromSeed = length(CellUV - SeedCenter);
float ForceAttenuation = exp(-DistFromSeed * ForceDecay) * ForceStrength;

// 反转版受力逻辑：
//   顺风方向 → 步长变小 → 衰减更慢 → 值更高 → 更容易传播
//   逆风方向 → 步长变大 → 衰减更快 → 值更低 → 更难传播
// 公式不变，因为 step 仍然是"代价"

float FMod_N  = max(0.2, 1.0 - dot(Dir_N,  ForceDir) * ForceAttenuation);
float FMod_S  = max(0.2, 1.0 - dot(Dir_S,  ForceDir) * ForceAttenuation);
float FMod_W  = max(0.2, 1.0 - dot(Dir_W,  ForceDir) * ForceAttenuation);
float FMod_E  = max(0.2, 1.0 - dot(Dir_E,  ForceDir) * ForceAttenuation);
float FMod_NW = max(0.2, 1.0 - dot(Dir_NW, ForceDir) * ForceAttenuation);
float FMod_NE = max(0.2, 1.0 - dot(Dir_NE, ForceDir) * ForceAttenuation);
float FMod_SW = max(0.2, 1.0 - dot(Dir_SW, ForceDir) * ForceAttenuation);
float FMod_SE = max(0.2, 1.0 - dot(Dir_SE, ForceDir) * ForceAttenuation);

// ============================================================
// 合并噪声 + 受力（加法混合）
// ============================================================
float Combined_N  = max(0.1, 1.0 + (NMod_N  - 1.0) + (FMod_N  - 1.0));
float Combined_S  = max(0.1, 1.0 + (NMod_S  - 1.0) + (FMod_S  - 1.0));
float Combined_W  = max(0.1, 1.0 + (NMod_W  - 1.0) + (FMod_W  - 1.0));
float Combined_E  = max(0.1, 1.0 + (NMod_E  - 1.0) + (FMod_E  - 1.0));
float Combined_NW = max(0.1, 1.0 + (NMod_NW - 1.0) + (FMod_NW - 1.0));
float Combined_NE = max(0.1, 1.0 + (NMod_NE - 1.0) + (FMod_NE - 1.0));
float Combined_SW = max(0.1, 1.0 + (NMod_SW - 1.0) + (FMod_SW - 1.0));
float Combined_SE = max(0.1, 1.0 + (NMod_SE - 1.0) + (FMod_SE - 1.0));

// ============================================================
// 反转传播：max(邻居 - step)
// 值从 1.0 向外递减，取最大的候选
// ============================================================
MaxCandidate = CurrentValue;

MaxCandidate = max(MaxCandidate, N  - BaseStep * Combined_N);
MaxCandidate = max(MaxCandidate, S  - BaseStep * Combined_S);
MaxCandidate = max(MaxCandidate, W  - BaseStep * Combined_W);
MaxCandidate = max(MaxCandidate, E  - BaseStep * Combined_E);
MaxCandidate = max(MaxCandidate, NW - BaseStep * ArcScale * Combined_NW);
MaxCandidate = max(MaxCandidate, NE - BaseStep * ArcScale * Combined_NE);
MaxCandidate = max(MaxCandidate, SW - BaseStep * ArcScale * Combined_SW);
MaxCandidate = max(MaxCandidate, SE - BaseStep * ArcScale * Combined_SE);

// 钳位到 [0, 1]
MaxCandidate = saturate(MaxCandidate);
```

## Stage 3：NormalizeGradient

```hlsl
// ============================================================
// NormalizeGradient (反转版)
// 输入：1.0 = 种子, 接近 0 = 最远
// 归一化：把 [MinReachedValue, 1.0] 重映射到 [0.0, 1.0]
// ============================================================

float RawValue;
Grid2DCollection.GetFloatValue<Attribute="GradientValue">(IndexX, IndexY, RawValue);

// 未到达区域保持 0
if (RawValue <= 0.0)
{
    Normalized = 0.0;
}
// 种子点
else if (RawValue >= 1.0)
{
    Normalized = 1.0;
}
else
{
    // MinGradientValue = 扩散到达最远处的值
    // 例如 BaseStep * GridSize/2 ≈ 1.0 时, 最远处 ≈ 0.0
    // 把 [MinGradientValue, 1.0] → [0.0, 1.0]
    Normalized = saturate((RawValue - MinGradientValue) / max(1.0 - MinGradientValue, 0.001));
}

// Gamma
Normalized = pow(Normalized, max(Gamma, 0.01));
```

## 方案局限性（为什么要换）

| 维度 | FloodStep 表现 | 流体真实 |
|---|---|---|
| 时间演化 | 每帧固定 step 推进 | 应有加速/减速/停止 |
| 质量守恒 | 无，种子点永恒 1.0 | 水会"流走" |
| 非线性反馈 | 无，step 只受预设 Noise/Force 影响 | 高度与流速互相耦合 |
| 扩散形态 | 各向异性测地距离场 → 同心等高线 | 应有冲出→摊开→边缘减速 |

简言之：FloodStep 算的是"到达时间"而不是"水位"，看起来像染料扩散但没有流体的物理反馈，所以视觉上死板、重复、缺生命感。

## 可复用的部分（移植到 SWE 方案时参考）

- **种子初始化逻辑**（Stage 1）：圆形 UV 距离判断 → SWE 里仍可复用，只是值从 1.0 改成高度高斯凸起
- **8 邻域噪声采样结构**：SWE 里如果要加各向异性扰动可以借用这套 `NMod_*` 结构
- **方向性受力模型**（ForceDirection/Strength/Decay + 指数衰减）：SWE 里作为初始速度场的方向参考
- **归一化 + Gamma**（Stage 3）：SWE 的 FinalHeightMap 导出前也需要这一步
