# assets — 全局共享资源

供**多个文档共用**的图片、图示、附件等二进制资源。

## 放什么

- 跨文档引用的图片（架构图、流程图、示意图）
- 多处复用的附件（PDF、压缩包等）

## 不放什么

- **只被单个项目引用**的图片 → 放到 `work/<项目名>/assets/` 或项目根下，和文档就近
- 截图暂存 → 先进 `inbox/`，确认要长期留再挪过来

## 组织建议

- 按大类分子目录：`assets/diagrams/`、`assets/screenshots/`、`assets/refs/`
- 文件名表达内容，不要 `image1.png` `未命名.png`
- 引用时用相对路径，便于工作区整体迁移
